## Sudoku Solver
A Python-based Sudoku Solver that uses a backtracking algorithm to efficiently solve Sudoku puzzles.

## Features
- Solves Sudoku Puzzle.
- Uses backtracking algorithm for efficient performance.
- Supports manual user input.

## Techology Used
Python
